# memoLearn
Memo Learn is a learning app that helps student digest information extracted from pdfs into flashcards. 

Below are the steps we recommend when accessing our working prototype:
1. Clone the git repository in local machine
2. Right click on the html document called ‘home.html’ and select ‘reveal in finder’ option
3. Once located in finder, open html document in preferred browser
4. Now the prototype is ready to function!
